# SCOUT

SCOUT is a YouTube creator discovery pipeline for creator sourcing. It starts from trusted seed channels, expands through adjacency signals, enriches candidate channels, ranks them with heuristic scoring, and adds keyword ingestion for direct discovery when the mention graph starts to saturate.

Stack: TypeScript, Cloudflare Workers, D1.
